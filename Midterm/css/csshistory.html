Version History

CSS Level 1 came in December 1996. This version was simply described as a formatting model for all the HTML tags.
CSS Level 2 became a recommendation by the W3C on 1998 as this version supported media-specific style sheets.
CSS Level 3 was recommended on 1999. This version's documentation was divided into modules and each module has a new extension.
