CSS Selectors

In CSS, selectors are used to select the elements that you wish to style.

Types of Selectors

  1. Simple selectors: selects element type, id, or class.
    2.     ◾ Example


  2. Attribute selectors: selects based on attributes or attribute values.
    3.     ◾ Example


  3. Pseudo-classes: selects elements that occurs in a specific state such as elements that are hovered or children of parent elements.
    4.     ◾ Example


  4. Pseudo-elements: selects elements in a specific position like first letters in a word.
    5.     ◾ Example


  5. Combinators: used to make more specific selections by combining two or more selectors.
    6.     ◾ Example


  6. Multiple selectors: not entirely a separate type but this is simply a combination or more that two selectors separated by a comma to perform multiple selections at once.
    7.     ◾ Example
