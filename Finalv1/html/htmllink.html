HTML Adding links and Image

HTML is using tags for its syntax. A tag is composed with special characters: <, > and /. They are interpreted by softwares to compose an HTML element.

To link text or an image to another web site, enclose the area you want to appear as a link with the < a href...tag > Example:
< a href="https://www.facebook.com/" > Facebook Home page < / a >
 will display as: Facebook Home page Be sure to close the tag with < /a >, otherwise everything that follows will be part of the link. To have the linked web page open in a new tab or window, add a target attribute Example:
< a href="https://www.facebook.com/" target="_blank">Facebook Home page< /a>

< img src="....." / > - Adds an image, either from a linked source or from the same source as the html page, depending on the url that follows.
