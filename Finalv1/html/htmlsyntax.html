HTML Syntax Tag

HTML is using tags for its syntax. A tag is composed with special characters: <, > and /. They are interpreted by softwares to compose an HTML element.

HTML Tags The most important characters in HTML are:     1. angle brackets (< >), which enclose HTML codes     2. forward slash (/), which is used to "close" HTML expressions HTML tags enclose the elements or sections of the web page they're intended to affect. Tags are differentiated -- and hidden -- from the content of the web page by the presence of angle brackets, i.e., < >. If I wanted to bold the word HTML in code it would look like HTML, or HTML Note that tags affecting a section of the text must be closed. This means adding a forward slash (/) before the tag is repeated at the end of the section.

VALUES Specific values are placed in quotation marks following the attribute. A value can be a specific size, color, font face, etc. Attributes and values can be applied to tags such as < font >, < img > (e.g., image height and width), < p > and many others, for example: ○ < font color="blue" > produces red text.     Output: produces red text. ○ < font size="+2"> doubles the size of the default text.     Output: doubles the size of the default text.

In summary, here's the syntax of a typical HTML expression:
< tag attribute="value" >
 Example:
< font size="+2" color="blue" >
font is the tag
size and color are attributes
"+2" and "blue" are values.
 And of course you must close the tag when you want those style characteristics to end, e.g.:
< font size="+2" color="Blue" >This is big BLUE text.< / font >
 Output:
This is big BLUE text.
Size values can be expressed as either relative (percentage, proportion) or absolute (a specific numerical value). In the example above, the "+2" means the font should appear at twice the size of the default.
