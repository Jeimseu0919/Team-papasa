HTML is using tags for its syntax. A tag is composed with special characters: <, > and /. They are interpreted by softwares to compose an HTML element.

HTML Tags The most important characters in HTML are:     1. angle brackets (< >), which enclose HTML codes     2. forward slash (/), which is used to "close" HTML expressions HTML tags enclose the elements or sections of the web page they're intended to affect. Tags are differentiated -- and hidden -- from the content of the web page by the presence of angle brackets, i.e., < >. If I wanted to bold the word HTML in code it would look like HTML, or HTML Note that tags affecting a section of the text must be closed. This means adding a forward slash (/) before the tag is repeated at the end of the section.

Document Structure Tags Web pages have two basic containers: the head and the body:     ○ The head tag, which contains information, or metadata, about the web page, such as a title for the page, javascripts, styles (CSS), meta information, and more. Whatever is inside the head tag does not display on the web page itself.     ○ The body tag, containing the content that actually displays in a browser

Horizontal Rule You can also separate sections with a horizontal line, as you see above, by using the hr tag. Because nothing is contained within it, hr is a self-closing tag. Best practice is to include the forward slash after the "hr," but most web browsers will display normally if you forget.
