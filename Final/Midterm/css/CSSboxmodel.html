CSS Box Model




















The box model illustrates the the contents that wraps around every HTML element. It contains the actual content, padding, border, and a margin.


  1. Margin is transparent and is a clear area around the border.
  2. Border could be set to be visible and surround padding and content.
  3. Padding is transparent and is around the content.
  4. Content is where text or images appear.
