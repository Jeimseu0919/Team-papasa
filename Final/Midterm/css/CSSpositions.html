CSS Positions

The position dictates the positioning method of an element.

  1. Static is the default position for elements. Elements set as static are not affected by left, right, bottom, and top properties.

  2. Relative sets an element to a normal position and it can be adjusted away from its position by left, right, bottom, or top.

  3. Fixed position locks an element in place even if the page is scrolled it will not move.

  4. Absolute positions an element to its neares positioned ancestor.

  5. Sticky swtiches between relative and fixed based on the scroll position.
